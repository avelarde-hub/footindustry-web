# ASERFINTEC - Food Industry Demo v4

Aplicación demo empresarial para la industria de alimentos, desarrollada en **React + Vite**, editable en **Visual Studio Code** y construida sin base de datos.

## Características

- Pantalla de acceso simulada
- Menú lateral desplegable
- Tema claro / oscuro
- Dashboard ejecutivo
- Módulos de producción, inventario, calidad, reportes y usuarios
- Roles demo: **Administrador** y **Usuario**
- Exportación local de reportes en **CSV** y **TXT**
- Uso del logo de **ASERFINTEC**
- Estructura modular basada en componentes y separación de responsabilidades

## Estructura del proyecto

```text
src/
  components/
  context/
  layout/
  modules/
  utils/
  assets/
```

## Ejecución

```bash
npm install
npm run dev
```

## Compilación

```bash
npm run build
```
